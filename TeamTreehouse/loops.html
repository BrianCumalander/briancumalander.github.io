While loop

    let counter = 0;
    while (counter < 10) {
        console.log(counter);
        counter += 1;
    }

For loop, here, you would normally use i as the counter, as in the Do Loop up above, but here, I break it down a little

        remember
    for ( decloration...let counter = 0; expresson...counter < 10; incrementer....counter += 1 )
     { do....console.log(counter); }

     such as:
     for ( let counter = 0; counter < 10; counter += 1 ) {
        console.log(counter);
     }
