JS Object Literals consist of the following pattern:
key: value,
property: value

    const student = {
        name: "Quincy",
        grades: [85, 90, 95, 100]
    };

Loop Through JS Objects

The format is as follows:

            for ( let key in object ) {
                //do something
            }

Lets say you have the following:

            const student = {
                name: 'Jesse',
                grades: [80, 85, 90, 95]
            };

            for (let key in student ) {
                //do something
            };

--'key' is the first part of your object. If you were to log the key to the console, as:

 for ( let key in student ) {
            console.log( key )
        };

It would output:

  1. name
  2. grades
Now, key is just a property name, so we could change it to propName, or whatever I like.
So how do we access the key values? We cannot do:

for ( let propName in student ) {
console.log( student.propName ); }

because it will litterally display 'propName' rather than the value of propName.
